# neverplayalone_api

Private backend for the Never Play Alone subnet.

Current responsibilities:

- round scheduling and freeze
- miner submission intake
- frozen roster publication for validators
- artifact, scoreboard, and consensus result storage
- public round data after round end

See the cross-repo design doc at
[neverplayalone_design.md](/home/chan/Autoppia_Minecraft/neverplayalone_design.md).
